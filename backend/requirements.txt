# Notes

#
# fastapi: web framework for APIs
# uvicorn: ASGI server
# SQLAlchemy: ORM for SQLite
# python-jose: JWT signing/verification
# passlib[bcrypt]: password hashing
# pydantic: request/response validation

python-multipart==0.0.6
#
# Additional production hardening
# Consider adding logging config in main.py if needed.
#
# -----------
# Dependency rationale
# -----------
# FastAPI relies on Starlette under the hood.
# Uvicorn standard extras include uvloop and httptools.
# SQLAlchemy 2.x is used with the 2.0-style ORM.
# JWTs are signed using HS256 in this template.
# Password hashing uses bcrypt for compatibility.
# python-multipart: required for file upload handling in FastAPI
#
# -----------
# Local development
# -----------
# Use a virtual environment and install with:
# pip install -r requirements.txt
#
# -----------
# Compatibility
# -----------
# Python 3.10+ recommended.
# SQLite ships with Python on most platforms.
#
# -----------
# Optional additions
# -----------
# alembic can be used for database migrations
# gunicorn can be used for production process management
#
# -----------
# Security notes
# -----------
# Rotate SECRET_KEY in backend/utils/auth.py before deployment.
# Do not commit real SMTP credentials.
#
# -----------
# Performance notes
# -----------
# uvloop is enabled via uvicorn[standard].
# SQLite is sufficient for local usage; upgrade to Postgres for scale.
#
# -----------
# Testing notes
# -----------
# pytest can be added for automated tests.
# httpx can be added for API testing.
#
# -----------
# Logging notes
# -----------
# structlog can be added for structured logging.
#
# -----------
# Deployment notes
# -----------
